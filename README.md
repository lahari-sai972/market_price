# 🌾 AgriPrice Pro

AgriPrice Pro is a web-based application developed to help farmers estimate crop market prices and receive crop suggestions based on their selected location and crop type. The application also supports multiple languages, making it more accessible to users.

## Features

- Crop Price Prediction
- Crop Suggestions
- State and City Selection
- User Login
- Contact Page
- Multilingual Support (English, Telugu, Hindi)
- Responsive User Interface

## Technologies Used

- React.js
- TypeScript
- Vite
- Tailwind CSS
- i18next
- React-i18next
- Lucide React

## Installation

Clone the repository:

```bash
git clone https://github.com/lahari-sai972/market_price.git
```

Navigate to the project folder:

```bash
cd market_price
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

## Language Support

The application supports the following languages:

- English
- Telugu
- Hindi

Users can switch between languages using the language selector available in the navigation bar.
